Job Descriotion Match Score Api

This is a RESTful API written in flask. ResumeData and Job description are send in the body of the Api with Post request returns an integer value

jobscore - Return an intger from 0 - 100

POST

/jobscore

Api Body - The body type is form-data
{
ResumeData : "I am Software Engineer having more than 2 years of experience",
JD: "We are looking for a C# developer having expereince in Micsoft Tech Stack"
}
Parameter Feild Type Description
1 ResumeData String This variable holds text of the resume as a string
2 JD String This variable holds text of the job description as a string
Success 200

Integer Value

Example Data on Success

36.71

Example with cURL
curl --location --request POST 'https://blooming-tor-44992.herokuapp.com/jobscore' \ --form 'ResumeData=I am Software Engineer having more than 2 years of experience ' \ --form 'JD=We are looking for a C# developer having expereince in Micsoft Tech Stack
